# Zomato Restaurant Expansion Analysis

## Project Overview
This project analyzes Zomato restaurant data to identify expansion opportunities, customer preferences, and business insights using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Creation
- Data Cleaning

## Key Analysis
- Restaurant distribution by city
- Cuisine analysis
- Ratings and reviews analysis
- Cost vs Rating comparison
- Expansion opportunity identification

## Files Included
- Excel Dashboard (.xlsx)
- Project Report (.docx)
- Supporting Documents
